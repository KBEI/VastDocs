Version 0.0.5

  • Removed CameraUtils utility class
  • Removed UrlValidator web Url validator
  • ScreenSizeUtils now no longer supports full screen judgment, only the length and width of the screen can be obtained
  • Deleted AspectRatioDevice , no longer supports defining more device types and their corresponding screen aspect ratios to judge full screen
  • MsgWindowUtils no longer supports popup Dialog
  • Added ColorUtils utility class
  • Added ShapeAndStateUtils utility class
  • Added User and Response beans
  • Optimized DateUtils utility class and fixed some logic
  • IntentUtils added resolveActivity judgment, and set prompt information for API 30 and above
  • Fixed the logical problem in the logPrint function of log printing, deleted the link that needs to add a Class, optimized the setting of the length judgment of log printing, and moved the LogContent interface to LogUtils.kt
  • MsgWindowUtils renamed to ToastUtils

Version 0.0.4

  • Delete NoMatchAspectRatio exception
  • Delete BuildVersionException exception
  • Delete 140 colors included
  • Added MergeBitmapUtils BitMap merge tool class
  • Added SystemUtils utility class
  • DateFormat adds common time formats and GMT time zone
  • DateFormatString GmtFormatString YearFormatString annotation is used to determine the parameter type in DateUtils
  • Improve DateUtils methods and annotations
  • For full screen judgment, isAllScreenDevice selects the corresponding API according to Build.VERSION.SDK_INT, isAllScreenDeviceApi31 , isAllScreenDeviceApi30 , isAllScreenDeviceApi30Down cannot be called from outside
  • The screen height is obtained by getMobileScreenHeight according to Build.VERSION.SDK_INT select the corresponding API, getMobileScreenHeightApi31 , getMobileScreenHeightApi30 , getMobileScreenHeightApi30Down be called from outside
  • Fix ScreenSizeUtils screen height acquisition problem

Version 0.0.3

  • Fixed the problem of incomplete log printing
  • Fix log printing stack information sometimes mismatch problem
  • Support to define more device types and their corresponding screen aspect ratios to judge the full screen

Version 0.0.2

  • Removed Adapter section

Version 0.0.1

  • Migrate the original code from jitpack to mavencentral, this version also includes the Adapter part
