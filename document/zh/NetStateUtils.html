NetStateUtil

编辑本页

借助 NetStateUtils 你可以很方便的查看一些关于你设备的网络状态

查看网络是否可用

// True为可用，否则为False
val available = isNetworkAvailable()

查看是否处于WIFI连接

// True为处于WIFI连接，否则为False
val isWifi = isWIFI()

查看是否处于手机移动网络

// True为处于手机网络连接，否则为False
val isMobile = isMobile()

查看WIFI信号强度

// signalStrength为信号强度
val signalStrength = getWifiDBM()
