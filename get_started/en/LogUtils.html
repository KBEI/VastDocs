LogUtils

Edit this page

Basic use

Here's how to print a info message

LogUtils.i(this.javaClass.simpleName,"Hello,This is a info")

Let's take a look at the running results

2022-02-08 21:34:48.024 9643-9643/com.gcode.vastutils I/class (MainActivity.kt:51): method: onCreate() key: MainActivity content: Hello,This is a info

By default，It shows the location of the print log MainActivity.kt:51 ，where to print the log method: onCreate() ，keywordkey: MainActivity ，and printed information content: Hello,This is a info.

Customize what to print

LogUtils provides an interface so that you can customize the format of content to be printed，for example:

// customize the format of content to be printed
LogUtils.setLogContentFormat(object : LogUtils.LogContent {
    override fun logContentFormat(methodName: String?, key: String?, content: String?): String {
        return "$methodName() $key $content"
    }
})
// Print log
LogUtils.i(this.javaClass.simpleName,"Hello,This is a info")

result:

2022-02-08 23:29:48.359 2836-2836/com.gcode.vastutils I/class (MainActivity.kt:46): onCreate() MainActivity Hello,This is a info

Printing super long text content is supported

LogUtils allows you to print super long content freely without restriction

var str = ""
repeat(2000){
    str += it.toString()
}
LogUtils.i(this.javaClass.simpleName, str)
